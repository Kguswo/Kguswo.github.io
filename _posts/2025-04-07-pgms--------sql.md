> ## 코드

```java
SELECT ID, NAME, HOST_ID
    FROM PLACES
        WHERE HOST_ID IN(
            SELECT HOST_ID
            FROM PLACES
            GROUP BY HOST_ID
            HAVING COUNT(*) >= 2
        )
            ORDER BY ID ASC;
```
