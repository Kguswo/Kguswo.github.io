<h5>문제</h5>

<p><code>CAR_RENTAL_COMPANY_RENTAL_HISTORY</code> 테이블에서 2022년 10월 16일에 대여 중인 자동차인 경우 '대여중' 이라고 표시하고, 대여 중이지 않은 자동차인 경우 '대여 가능'을 표시하는 컬럼(컬럼명: <code>AVAILABILITY</code>)을 추가하여 자동차 ID와 <code>AVAILABILITY</code> 리스트를 출력하는 SQL문을 작성해주세요. 이때 반납 날짜가 2022년 10월 16일인 경우에도 '대여중'으로 표시해주시고 결과는 자동차 ID를 기준으로 내림차순 정렬해주세요.</p>

> ## 문제 풀이

case문으로 조건에 맞는 경우를 각각 출력하고자 했다. 이때 집계함수 사용전에는 그냥 groupby한 후 이를 case문으로 작성했는데 틀렸다. 이에 집계함수 MAX로 조건에 맞으면 1, 맞지 않으면 0을 만들었고 이렇게 하니까 정답처리가 되었다.

전체 과정은 
 1. CAR_ID로 GROUP_BY해서 중복제거
 2. 특정 날짜를 MAX() 집계함수로 조건문 통해 1, 0 반환
 3. CASE문으로 출력문 조정
 4. 별칭 지정 후 내림차순정렬

> ## 코드

```sql
SELECT DISTINCT CAR_ID, 
    CASE
        WHEN MAX('2022-10-16' BETWEEN START_DATE AND END_DATE) = 1 THEN '대여중'
        ELSE '대여 가능'
    END AS AVAILABILITY
        FROM CAR_RENTAL_COMPANY_RENTAL_HISTORY
            GROUP BY CAR_ID
                ORDER BY CAR_ID DESC
```
